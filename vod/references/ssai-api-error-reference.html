일반 오류 메시지

상태 이름 메시지 원인
400 BAD_REQUEST Invalid JSON body on request 요청 본문에 대한 JSON에 오류가 있습니다. 기본 JSON 문제이거나 필드 값의 잘못된 유형일 수 있습니다.
400 BAD_REQUEST "error": "{fieldname} field is required 요청 본문에 표시된 필수 필드가 누락되었습니다.
400 AD_CONFIG_NOT_FOUND Ad config not found SSAI 요청에 지정된 광고 구성을 찾을 수 없습니다.
400 AD_CONFIG_INACTIVE Ad config is inactive SSAI 요청에 지정된 광고 구성이 비활성 상태입니다.
401 UNAUTHORIZED Permission denied. OAuth 액세스 토큰이 없거나 잘못되었습니다. 액세스 토큰을 제공 한 경우이 요청에 적절한 범위가 없을 수 있습니다.
403 NOT_AVAILABLE Resource is not available. 요청한 리소스를 일시적으로 사용할 수 없습니다. 비디오 처리가 진행되는 동안 일시적인 상태 일 수 있지만 메시지가 지속되면 지원팀에 문의하세요.
404 NOT_FOUND Resource does not exist. 존재하지 않는 리소스를 요청하셨습니다. 광고 구성을 업데이트하거나 더 이상 존재하지 않는 세션에서 광고 추적을 가져 오려고 할 수 있습니다. 이 오류는 엔드 포인트에서 허용되지 않는 HTTP 메서드를 사용하는 경우에도 반환됩니다.
500 UNKNOWN an unknown internal error occurred Brightcove 시스템의 문제-나중에 다시 시도하십시오.
503 SERVICE_UNAVAILABLE The API is temporarily unavailable 백엔드 문제-나중에 다시 시도하세요.
504 TIMEOUT something took too long 서버 사용량이 너무 많을 수 있습니다. 나중에 다시 시도하세요.

개요

SSAI (Server-Side Ad Insertion)를 사용하면 동영상에 광고를 삽입하여 브라우저의 광고 차단기로 광고를 차단할 수없는 TV와 같은 시청 환경을 제공 할 수 있습니다. SSAI의 경우 차세대 수집 및 전송 시스템인 Dynamic Delivery를 사용하여 비디오를 수집해야 합니다. Dynamic Delivery 는 스토리지 공간을 줄이고 미디어를 동적으로 패키징합니다.

SSAI는 DRM 및 비 DRM 콘텐츠 모두에서 작동합니다.

메모:
  • SSAI를 사용하려면 Video Cloud 계정을 다음과 같이 구성해야합니다. 동적 전달 SSAI에 대해 활성화됩니다. 이 기능을 사용하려면 계정 관리자에게 문의하세요.
  • SSAI는 VOD (Video-on-Demand)와 함께 작동하며 라이브 재생을 지원합니다.
  • 제한 사항 목록은 SSAI 개요문서를 참조하십시오.

Dynamic Delivery로 수집 된 Video Cloud에서 검색하는 모든 비디오에는 광고 구성의 VMAP 파일에 지정된 광고가 포함됩니다.

Video Cloud에 저장된 비디오 콘텐츠로 서버 측 광고를 재생하려면 다음 단계를 따르십시오.

  1. 광고 태그 정의
  2. 광고 태그 테스트
  3. 광고 구성 만들기
  4. 비디오 요청
  5. 플레이어 구성

다음은 몇 가지 추가 주제입니다.

  • 광고 태그 변수
  • 서버 측 비 커닝

광고 태그 정의

광고 태그는 검색 및 재생 될 광고를 정의합니다. 다음 형식 중 하나를 사용할 수 있습니다.

  • 디지털 동영상 광고 게재 템플릿 (VAST) -하나의 광고를 정의합니다.
  • VMAP (Video Multiple Ad Playlist) -광고의 재생 목록을 정의합니다. VMAP 파일은 VAST 태그 모음입니다.

광고 태그 테스트

광고 태그가 유효하고 예상대로 작동하는지 확인하는 것이 좋습니다. 이는 구현 프로세스 중에 다른 문제를 디버깅하는 데 도움이됩니다.

  • Google의 Video Suite Inspector

광고 구성 만들기

광고 구성은 VMAP (Video Multiple Ad Playlist) 파일, 비콘 및 기타 구성에 대한 URL을 포함하여 SSAI 재생의 다양한 측면을 정의합니다.

Video Cloud Studio 사용

Studio를 사용하여 광고 구성을 만들려면 서버측 광고 설정구성문서를 참조하십시오.

SSAI Ad Config API 사용

API를 사용하여 광고 구성을 만들려면비디오 클라우드 SSAI 광고 구성 API문서를 참조하십시오.

비디오 요청

다음으로 서버 측 광고로 비디오 데이터를 가져옵니다. 이렇게하려면 광고 구성 ID를 Playback API 동영상 요청에 URL 매개 변수로 추가하세요.

VMAP 응답에서 브라이트코브는<vmap:Extensions>섹션을 추가합니다.

확장 섹션의<bc:Brightcove>태그와 연결된 속성은 다음과 같습니다.

VMAP 응답 필드 속성
속성 기술
contenturi 세션 별 매니페스트에 대한 참조-플레이어에로드해야하는 항목입니다.
contentlength 콘텐츠 비디오의 길이
payloadlength 연결된 총 스트림의 길이

광고 시간

다음 속성은 각 광고 블록과 연관됩니다.

VMAP 응답 필드
속성 기술
timeoffset 계산해야 하는startend , 또는 특정 시간일 수 있습니다. 포드에 여러 개의 광고가있는 경우 동일한 "콘텐츠 시간"에 삽입되므로 동일한 오프셋을 가질 수 있습니다.

다음 태그는 각 광고 블록과 연결되어 있습니다.

메모

광고 전략을 만들 때 유의해야 할 사항은 다음과 같습니다.

  • start오프셋에서 광고 지속00:00:16.0000시간이 3개인 경우 총 프리롤 포드 길이가 48초가 될 수 있습니다. 이는 실제 콘텐츠 시작 시간이 실제 연결 매니페스트의 48 초 표시가됨을 의미합니다.
  • 다양한 광고 요소 중에서 가장 관련성이 높은 것은 아마도 활성화할 URL 링크를 식별하는 것입니다. ClickThrough 사용자가 링크를 클릭할 때 연결된 모든ClickTracking요소를 신호로 표시해야 합니다.

광고 태그 변수

광고 태그 변수를 사용하면 광고를 타겟팅 할 수 있습니다.

광고 구성에{% raw %}{{url.*}}{% endraw %}매크로가 있는 경우 VMAP을 요청할 때 쿼리 매개 변수로 추가해야 합니다.

예를 들어, 다음과 같은 매크로

{% raw %}{{url.deviceId}}{% endraw %}

다음과 같은 쿼리 매개 변수로 추가되어야합니다.

?deviceId=xyz

자세한 내용은비디오 클라우드 SSAI 광고 구성 API문서를 참조하십시오.

서버 측 비 커닝

서버 측 노출 및 사 분위 / 완전 비콘을 실행하도록 비 커닝을 구성 할 수 있습니다.

  • SSAI 사용시 VAST 광고 오류 비콘은 광고 워크 플로의 문제를 사전에 찾고 수정하는 데 도움이 될 수 있습니다. 자세한 내용은 SSAI가 포함된광고 오류 비콘문서를 참조하십시오.
  • VAST 표준 비콘도 사용할 수 있습니다. 자세한 내용은디지털 비디오 광고 게재 템플릿 (VAST)문서를 참조하십시오.

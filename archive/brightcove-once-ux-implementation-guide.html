VMAP 광고 집계

NS 위의 콘텐츠 변수 제공 For Once UX는 서비스의 Ad Aggregation 방식을 활용합니다. 이 메소드는 요약 또는 집계 구성된 applicationguid 및 광고 서버를 기반으로 한 광고 보기 데이터. Once UX 응답을 수신 할 때이 메소드는 VAST 3.0으로 정규화 된 후 VMAP 1.0 XML 객체에 삽입 된 전체 자산의보기 세션에 대한 광고로드를 포함하는 메타 데이터 및 VMAP 객체를 제공합니다. 이 VMAP 응답은 사용자 경험 (광고 크리에이티브 또는 광고 포드 카운트 다운, 플레이어 컨트롤 숨기기), 비 커닝 및 추가 크리에이티브 유형 (배너, 클릭 연결 및 VPAID 광고)과 관련된 클라이언트 측 광고 기능을 처리 할 수있는 기능을 제공합니다. 이 VMAP 개체에는 Brightcove Once 데이터와 재생에 사용할 광고 삽입 된 Once URL이 포함 된 Brightcove Once XML 개체도 포함되어 있습니다. 자세한 내용은 VMAP 샘플 응답 예제를참조하십시오.

동적 광고 매개 변수

일단 UX는 광고를 채우는 데 허용되는 모든 DAP (동적 광고 매개 변수)를 허용하고 Once 플랫폼과 동일한 비콘 호출을 호출합니다. 이러한 값에 대한 설명은 한 번시작하기 를 참조하십시오. 동적 광고 매개 변수 .

쿼리 문자열 매개 변수

일단 UX가 Once 플랫폼과 동일한 쿼리 문자열 매개 변수를 활용하면이 섹션에서는 Once UX 응답 및 동작의 추가 미세 조정을 지원하는 데 사용할 수있는 세 가지가 있습니다. 한 번시작하기 를 참조하십시오. 사용 가능한 다른 쿼리 문자열 매개 변수에대한 자세한 내용은 쿼리 문자열 매개 변수를 참조하십시오.

추적 이벤트 비활성화

UMTP= {integer}

UMTP는 기본 브라이트코브 서버측 광고 비디오 노출 및 이벤트 추적 동작을 재정의하거나 비활성화하는 데 사용됩니다.

  • UMTP = 0

기본적으로 Brightcove는 클라이언트 / 애플리케이션을 대신하여 광고 서버에 선형 광고 비디오 노출 및 추적 이벤트를 실행합니다. 매개 변수가 명시 적으로 설정된 경우 Once는 노출 또는 추적 이벤트를 실행하지 않습니다.

산출

출력= {integer}

이 키-값 쌍은 Brightcove가 반환하는 응답의 개체 형식을 제어합니다. 기본적으로 Once UX는 XML로 응답하므로 JSON 응답이 필요한 경우 output = 1을 사용해야합니다.

  • XML = 0
  • JSON = 1

  • dfxp
